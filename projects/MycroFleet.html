MycroFleet Sensor Network

Overview

MycroFleet is an affordable sensor fleet for 1–5 acre farms. It focuses on simple deployment, robust data, and long battery life.

My role

  • Firmware (FreeRTOS, low-power, OTA-ready architecture)
  • PCB bring-up & instrumentation
  • FastAPI microservice for data ingest

Architecture

ESP32 nodes → Gateway → FastAPI → Timeseries DB (e.g., Influx/Timescale) → Grafana dashboards.

Highlights

  • Sub-$25 BOM per node (target)
  • LoRa/2.4 GHz variants
  • Field-safe OTA staging
